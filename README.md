# 🍱 Food Share Platform

## 📌 Project Description
Food Share Platform is a web application that helps reduce food wastage by connecting people who want to donate food with those who need it.

This system allows users to add food donations, view available food, and request food easily.

---

## 🎯 Objective
- Reduce food wastage
- Help needy people
- Connect donors and receivers
- Build a simple full-stack application

---

## 🚀 Features
- Add food donations
- Request food
- View available food items
- Simple and user-friendly interface
- Frontend + Backend integration
